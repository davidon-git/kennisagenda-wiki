# Kennisagenda Wiki — Provincie Utrecht 2026-2027

Interactieve beleidswiki gebaseerd op de **Strategische Kennisagenda 2026-2027** van de Provincie Utrecht.

## Wat is dit?

Een visuele, klikbare wiki die pagina 2 van de kennisagenda (het mozaïek) gebruikt als interface. Per thema vind je AI-gegenereerde wikipagina's met bronverwijzingen uit het Nederlandse beleidscorpus (CPB, PBL, SCP, WRR, RLI, RIVM, SER, en meer).

## Thema's

- 🌿 Kwaliteit van leven (brede welvaart)
- 👥 Verbonden samenleving
- 📊 Demografie
- 💻 AI en digitalisering
- ☀️ Toekomstbehendigheid
- 🌍 EU en internationale ontwikkelingen

## 29 artikelen, 89+ bronnen

Elk artikel is voorzien van 4–5 bronverwijzingen uit verschillende instituten. Alle bron-URLs zijn geverifieerd (HTTP 200).

## Live bekijken

## Techniek

- Pure HTML/CSS/JS — geen frameworks
- Hash-based routing (werkt out-of-the-box op GitHub Pages)
- Inline SVG-iconen (geen externe afhankelijkheden)
- Data geladen uit `wiki-data.js`
